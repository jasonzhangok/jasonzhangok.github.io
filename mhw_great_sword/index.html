怪物猎人世界大剑总结

发表于 Aug 11 2019 | 分类于 Game
怪物猎人世界的大剑操作总结。

怪物猎人世界的大剑操作总结，根据B站视频及网上技术贴整理。

攻击技能

  • 三角/Y/鼠标左键：直斩，可蓄力
  • 圆圈/B/鼠标右键：横扫斩，范围广威力低，除刷小怪可无视的攻击方式
  • 竖斩后再次三角/Y/鼠标左键：强蓄力斩
  • 强蓄力斩后再次三角/Y/鼠标左键：真蓄力斩
  • 竖斩蓄力同时圆圈/B/鼠标右键：肩撞，可接强蓄力斩
  • 强蓄力斩蓄力同时圆圈/B/鼠标右键：肩撞，可接真蓄力斩
  • 快速两下三角/Y/鼠标左键：直斩后拍击，可无视
  • 圆圈/B/鼠标右键+三角/Y/鼠标左键：上捞斩，可无视
  • 持刀状态翻滚后三角/Y/鼠标左键：肩撞
  • 加速下坡时短按三角/Y/鼠标左键后松开：强力上捞斩，人会飞起，用于骑乘
  • 强力上捞斩后三角/Y/鼠标左键：下坠刺击，多段判定伤害

核心

肩撞取消强蓄力斩，直接出真蓄，是大剑输出的核心部分。

速真蓄连招

  1. Y直斩(不蓄力)→Y强蓄圆圈取消(肩撞)→真蓄
  2. Y蓄力直斩B取消(肩撞)→Y强蓄B取消(肩撞)→真蓄
  3. 翻滚后Y(肩撞)→Y强蓄B取消(肩撞)→真蓄（最快进入真蓄的方法）
    注意，肩撞取消必须是在Y按住蓄力的情况下按B，不然很容易变成捞斩。

Tips

  1. 肩撞消耗耐力。肩撞期间有霸体并减伤50%，不会被怪物招式打飞，可用于硬钢各种怪物铁山靠、扫尾、龙车等攻击。这也是我们能莽真蓄的资本。用好了还能无视怪物吼。
  2. 真蓄斩有两段，人会翻个360度的跟斗，并且前移一段距离，请算好距离，不然会飞过头砍不到怪。第一段伤害很低，只有第二段的10%左右。
  3. 虽然名字叫真蓄力斩，但并不是一定要蓄力。其实蓄力三段的伤害也就是直接砍出去的1.2倍差不多，具体倍率忘了，反正很多时候多贪一段蓄力意味着砍空。
  4. 真蓄斩砍出后硬直极长，但和别的出招一样都可以用翻滚取消硬直。
  5. 直斩(Y)、上捞斩(Y+B)、横扫(B)可以无缝(不存在硬直)衔接，并且顺序无关，新手“瞎打”招式。
  6. 蓄力是可以通过摇杆小幅度地改变砍下来的方向，向左或向右推摇杆，可以有大约30度的方向变更。

防御模式

按下LT进入防御模式，防御模式下可以挡住包括吼叫在内的怪物大部分招式，以武器锋利度极度下降为代价。收刀状态下瞬间按下LT可以发动瞬防。

防御模式下(按下LT)，再按下三角/Y/鼠标左键，就会踢。可以用来踢醒睡眠或晕眩状态的队友。

防御模式下按一次三角/Y/鼠标左键，就是踢，踢之后再按三角/Y/鼠标左键就是肩撞(铁山靠)。

空战大剑

在可攀爬墙壁上进行反跳，滞空时长按三角/Y/鼠标左键，就可以进入超速蓄力，伤害略高于普通蓄力，可对怪物附加极高的骑乘值。

滑坡大剑

滑行状态下按三角/Y/鼠标左键，就可以进入超速蓄力，看准时机松开就可以起跳，落地时会使出下坠突刺 并附加一小段的骑乘值，落地之后会衔接一记强横扫斩。找好起跳机会和下落落点。

注意：

  • 划过头就不会起跳了，变成了普通的蓄力斩；
  • 滑坡不能再持武器状态下机型，必须收刀模式。

铁山靠(肩撞)的三种使用方法

  1. 蓄力状态下按下圆圈/B/鼠标右键；
  2. 防御模式下按两下三角/Y/鼠标左键；
  3. 翻滚后按下三角/Y/鼠标左键。

推荐招式

  1. 三角/Y/鼠标左键直斩，然后马上翻滚收刀，然后走位找怪物破绽。
  2. 站桩输出招式：Y→Y+B→Y→Y+B→……，位置不会有任何变化，招式之间不会有任何硬直。
  3. 强蓄力斩+强横扫斩的无缝衔接：强蓄模式下按下Y打出一个强蓄力斩，然后再按B就可以接一个强横扫，并且在强横扫后再按Y就又能进入强蓄力状态，不需要回去蓄普通蓄力。强横扫继承前一个强蓄力斩的蓄力等级。
  4. 铁山靠后接飞身跃入斩：铁山靠后按下B就可以接飞身跃入斩。与强蓄力斩+强横扫斩类似，飞身跃入斩也是继承上一个蓄力等级，所以在蓄力状态下按B使出铁山靠，而不是直接翻滚后接Y进入蓄力状态。
  5. 眠斩：怪物被唤醒的拿一下伤害是翻倍的，用真蓄力斩最后一刀去唤醒怪物，找合适的距离让蓄力的三招打空，保证真蓄的最后一击命中怪物。方法：贴住怪物的脸，然后按RT拔刀，然后后翻滚，翻滚后的位置不要移动，再进行一次不移动的转身，最后打出普蓄、强蓄、真蓄，最后一下一定命中怪物。

注意：

  • 飞身跃入斩有极高的武器属性补正，所以使用带属性的武器会大大提高伤害；
  • 飞身跃入斩容易把怪打失衡。例如火龙，对腿打两次铁山靠+飞身跃入斩就可以把它打倒地，倒地后再对头进行铁山靠就可以把它打晕，可以利用这一点无限击倒击晕。
